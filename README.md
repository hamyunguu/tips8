# Book Atlas®

An interactive WebGL experience to explore independent Korean publishers' books —
a reskin of the [Album Atlas](https://albumatlas.vercel.app) concept for books.

Search an author, then drift through their books as cover plates scattered across a
pannable canvas. Each cover bends and splits into RGB channels as you drag (the
signature velocity-warp + chromatic-aberration shader). Click a book for its
details; click the background or **Return** to drift back.

**Data:** scraped catalogs of [workroom press](https://workroompress.kr) (40 books,
real authors) and [6699press](https://6699press.kr) (99 design works) — 139 books in
all. Covers are loaded directly from the publishers' CDNs.

## Stack

- **SvelteKit** (Svelte 5) — fully static (`@sveltejs/adapter-static`), no server
- **Three.js** — the album/book atlas + custom GLSL shaders
- **anime.js v4** — UI motion
- **Tailwind CSS v4** — theme tokens, `PP Neue Montreal`

## Develop

```bash
npm install
npm run dev      # http://localhost:5175
```

## Build

```bash
npm run build    # static site → ./build
npm run preview
```

## Data

`src/lib/data/books.json` is generated by the scrapers (run in order):

```bash
node scripts/scrape-workroom.mjs   # workroom press → books.json
node scripts/scrape-6699.mjs       # merges 6699press in
```

## Deploy

Pushing to `main` builds and publishes to GitHub Pages via
`.github/workflows/deploy.yml`.
